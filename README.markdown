TUnit-NetBeans MGL7361
=============

Suite de tests TUnit sur NetBeans 7

Prérequis
---------

Java [JDK >= 6][jdk], NetBeans 7 [netbeans] et [git][git] doivent être installés.

Marche à suivre
---------------

    $ git clone git://github.com/gitalpha/TUnit-NetBeans.git
